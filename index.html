Projects

  • CAS - Central Authentication Service
    The project CAS at Cedar Gate is the customized form of Apereo CAS which is an open-source Java server component. It supports Single Sign-On (SSO) functionality across different products and services introduced by Cedar Gate. As the name suggests, it is mainly used for authentication, but not limited. CAS supports username-password login and SAML login along with multi factor authentication. It is also used for Clickwrap Agreement, SAML Authorization, Proxy Authentication, Maintaining different password policies along with an ability to determine compromised and commonly used passwords and many more.
  • UM - User Management
    User Management (UM) is a web application which is used to perform user related operations. It provides an ability to create/edit/disable user(s), grant/revoke client and service level access to user(s), generate different kinds of user related reports (Manually via the UI/Automatic via the CRON), add/edit client, add/edit/delete service/role/permission along with different level of access that is required across different products of Cedar Gate.
  • Security Core Plugin
    Security Core Plugin is a common plugin developed under grails 2.5.1, 3.1.4, 3.3.8 and 4.0.10 and used by all the products of Cedar Gate as a part for authorization and querying user related records from database. It holds all the user related domains maintained for adopting HQL and Hibernate queries.
  • Security Plugin
    Security Plugin is a common plugin used for authentication which is developed under grails 2.5.1, 3.1.4, 3.3.8 and 4.0.10 and used by all the products of Cedar Gate. It is developed on the top of Spring Security which uses Spring Security Core and Spring Security CAS to authorize the user that was authenticated by CAS at the prior stage. This plugin is used as a communicating agent between CAS and the application that uses the plugin.
  • Deertrack Plugin
    Deertrack Plugin is a common plugin which is used for tracking user activity. Similar to all other plugins, it is also developed under grails 2.5.1, 3.1.4, 3.3.8 and 4.0.10 and is used by all the products of Cedar Gate. It stores information about the pages visited by the user during his/her active session along with the browser details, start-end time and total time spent on a single page. These data are later used for generating different kinds of reports in UM.
  • Hamro College
    An Android based application program which is helpful to students, teachers and any other college staffs who are associated with the academic purpose of the college. This application can take attendance, send notifications, view daily class routine, calculate final internal marks and many more college activities. This application was developed as a Major Project where I contributed on some portions of UI design, Report Development and Data Collection.
  • Optical Character Recognition Using Neural Network
    A program written in Python language which scans the image having Devanagari script and converts in into machine encoded text form.
